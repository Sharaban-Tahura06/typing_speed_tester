# ⌨️ Typing Speed Tester

A simple terminal-based **Typing Speed Tester** built with Python.
Test how fast and accurately you can type!

---

## 🚀 Features

- 📝 Random sentence generator
- ⏱️ Real-time WPM (Words Per Minute) calculation
- 🎯 Accuracy percentage checker
- 🏆 Leaderboard to track your best scores
- 💾 Scores saved locally using JSON
- 🔁 Replay option after each test

---

## 🛠️ Built With

- Python 3.x
- Built-in modules only (`time`, `random`, `json`, `os`)
- No external libraries required!

---

## ⚙️ How to Run

1. Clone the repository:
```bash
   git clone https://github.com/Sharaban-Tahura06/typing_speed_tester\
```

2. Go into the project folder:
```bash
   cd typing_speed_tester
```

3. Run the program:
```bash
   python main.py
```
